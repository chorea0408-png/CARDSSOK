# 카드쑉 서비스 설명서

> 원고만 있으면, 카드뉴스는 카드쑉이 만들어드려요.

---

## 이런 고민 있으신가요?

- SNS에 카드뉴스를 올리고는 싶은데, 디자인 툴을 배우기엔 시간이 없다
- 미리캔버스나 캔바를 켜봤지만 템플릿이 너무 많아서 뭘 골라야 할지 모르겠다
- 회원가입하고 로그인하는 것부터가 귀찮다
- 급하게 하나 뽑아야 하는데 30분씩 디자인 만지고 있을 여유가 없다

카드쑉은 이 모든 고민을 "원고를 붙여넣는다"는 동작 하나로 해결합니다.

---

## 카드쑉이 뭔가요?

**원고 텍스트를 붙여넣으면, 카드뉴스 슬라이드가 자동으로 만들어지는 무료 웹 서비스**입니다.

회원가입도, 로그인도, 템플릿 고르기도 필요 없습니다. 브라우저에서 바로 열고, 글을 붙여넣으면 끝입니다.

---

## 핵심 특징 4가지

### 1. 붙여넣으면 바로 완성돼요
글을 문단 단위(엔터 두 번)로 붙여넣기만 하면, 각 문단이 자동으로 슬라이드 한 장씩으로 나뉘어요. 첫 줄은 제목, 나머지는 본문으로 알아서 배치됩니다. 버튼 하나 더 누를 필요 없이, 붙여넣는 순간 바로 결과물이 눈앞에 뜹니다.

### 2. 어떤 글이든 어울리는 디자인을 알아서 골라줘요
"할인", "이벤트" 같은 단어가 많으면 화사한 톤으로, "일상", "에세이" 같은 단어가 많으면 차분한 톤으로 — 내용에 맞는 색감과 폰트를 자동으로 추천해드려요. 디자인 감각이 없어도 걱정 없습니다.

### 3. 마음에 드는 스타일을 원클릭으로
그래도 직접 꾸미고 싶다면, "모던 블루" · "다크 시크" · "피치 감성" · "미니멀 화이트" 중 하나만 눌러도 전체 슬라이드의 톤이 한 번에 바뀝니다. 물론 색상, 폰트, 그림자, 배경까지 세세하게 직접 조정하는 것도 가능해요.

### 4. 완성한 건 바로 들고 나갈 수 있어요
PNG, JPG, ZIP(전체 슬라이드 한 번에), JSON(재편집용) 등 원하는 형태로 바로 다운로드하세요. 화질도 1x/2x/3x 중에 골라서 인쇄용부터 SNS 업로드용까지 대응합니다.

---

## 이렇게 사용해요 (3단계)

1. **원고 입력** — 미리 써둔 글을 붙여넣거나, 그 자리에서 직접 입력하세요. ChatGPT나 Claude로 먼저 초안을 만들어 와도 좋아요.
2. **자동 분리 & 디자인** — 슬라이드가 자동으로 나뉘고, 원하면 원클릭 스타일이나 세부 색상/폰트로 다듬으세요.
3. **내보내기** — PNG/JPG/ZIP 중 원하는 형식으로 다운로드해서 바로 SNS에 올리세요.

---

## 이런 분들께 추천해요

| 사용자 | 이렇게 씁니다 |
|---|---|
| SNS 마케터 / 콘텐츠 담당자 | 매일 올리는 카드뉴스, 원고만 준비되면 몇 분 안에 초안 완성 |
| 소상공인 / 자영업자 | 디자이너 없이도 이벤트·공지 카드뉴스를 직접 뚝딱 |
| 블로거 / 인플루언서 | 긴 글을 카드뉴스로 재가공해서 다른 채널에도 재활용 |
| 학생 / 동아리 운영진 | 발표자료, 홍보물을 급하게 만들어야 할 때 |

---

## 다른 서비스와 뭐가 다른가요

미리캔버스, 캔바 같은 도구들은 훌륭하지만 "템플릿을 고르고 → 배치를 만지는" 과정이 필요합니다. 카드쑉은 그 과정 자체를 생략했습니다.

| | 카드쑉 | 일반 디자인 툴 |
|---|---|---|
| 로그인 | 필요 없음 | 대부분 필요 |
| 시작하기까지 | 원고만 붙여넣으면 끝 | 템플릿 탐색 → 편집 |
| 비용 | 완전 무료 | 무료+유료 티어 혼합 |

대신 스톡 이미지나 화려한 템플릿 라이브러리는 없습니다. **"빠르게, 로그인 없이, 초안 하나 뽑는 것"**에 집중한 도구예요.

## 자주 묻는 질문

**정말 완전 무료인가요?**
네. 광고를 통해 서비스를 운영하고 있어서, 사용자에게 별도 비용을 받지 않습니다.

**만든 카드뉴스는 어디에 저장되나요?**
회원가입이 없기 때문에, 작업 내용은 지금 사용 중인 브라우저에만 저장됩니다. 다른 기기에서는 이어서 작업할 수 없고, 브라우저 캐시를 지우면 사라지니 완성한 결과물은 미리 다운로드해두세요.

**모바일에서도 되나요?**
네, 모바일 웹에서도 원고 입력부터 디자인, 내보내기까지 전부 가능합니다.

**만든 이미지를 상업적으로 써도 되나요?**
직접 만든 텍스트 기반 디자인이라 자유롭게 사용하실 수 있어요. 다만 별도로 업로드한 이미지가 있다면 그 이미지의 저작권은 사용자 책임입니다.
